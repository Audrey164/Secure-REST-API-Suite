# Secure REST API Suite

A highly secure REST API system with JWT, OAuth2, rate limiting, and Dockerized microservices.

## Features
- JWT authentication  
- OAuth2 compatibility  
- Rate limiting  
- Input sanitization  
- API versioning  

## Stack
- Laravel API  
- MySQL  
- Redis  
- Docker  

## Installation
```bash
composer install
php artisan migrate
php artisan serve
